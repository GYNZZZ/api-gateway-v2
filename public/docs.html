API Reference

API Gateway 开发者文档

通过 OpenAI-compatible 接口访问模型，同时使用统一的 API Key、余额和日志管理能力。

Base URLhttp://localhost:3000
Authentication

身份鉴权

所有用户接口都需要在请求头中携带用户 API Key。

HTTP Header
Authorization: Bearer <user-api-key>
不要把 API Key 放在查询参数、公开仓库或客户端日志中。
POST

/v1/chat/completions

发送 OpenAI-compatible 聊天请求。成功响应后会扣除当前账户余额并记录调用日志。

请求示例

JSON
{
  "model": "gpt-4.1-mini",
  "messages": [
    { "role": "user", "content": "Hello" }
  ]
}

返回示例

{
  "id": "chatcmpl-example",
  "object": "chat.completion",
  "model": "gpt-4.1-mini",
  "choices": [{
    "index": 0,
    "message": { "role": "assistant", "content": "Hello!" },
    "finish_reason": "stop"
  }]
}
GET

/v1/me

查询当前 API Key 所属用户的基本信息与余额。

请求示例

GET /v1/me
Authorization: Bearer <user-api-key>

返回示例

{
  "id": 1,
  "name": "test_user_1",
  "balance": 100
}
GET

/v1/logs

返回当前用户自己的调用日志，不会返回其他账户的数据。

请求示例

GET /v1/logs
Authorization: Bearer <user-api-key>

返回示例

{
  "total": 1,
  "data": [{
    "route": "/v1/chat/completions",
    "status": 200,
    "charged": 1,
    "durationMs": 12
  }]
}
Administration

管理员接口

管理员接口使用单独的请求头，不应暴露给普通用户。

x-admin-api-key: <admin-api-key>
GET/admin/users

获取全部用户列表

GET/admin/logs

获取全部调用日志

Errors

错误码说明

401

未授权

API Key 缺失或无效。

402

余额不足

账户额度不足以完成调用。

429

请求过多

为后续限流能力预留。

500

服务器错误

服务内部发生异常。

502

上游错误

上游服务失败或不可用。

Code examples

语言示例

CMD
curl.exe -X POST http://localhost:3000/v1/chat/completions ^
  -H "Authorization: Bearer user-key-001" ^
  -H "Content-Type: application/json" ^
  -d "{\"model\":\"gpt-4.1-mini\",\"messages\":[{\"role\":\"user\",\"content\":\"Hello\"}]}"
JavaScript
const response = await fetch("http://localhost:3000/v1/chat/completions", {
  method: "POST",
  headers: {
    "Authorization": "Bearer user-key-001",
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    model: "gpt-4.1-mini",
    messages: [{ role: "user", content: "Hello" }]
  })
});
console.log(await response.json());
Python
import requests

response = requests.post(
    "http://localhost:3000/v1/chat/completions",
    headers={"Authorization": "Bearer user-key-001"},
    json={
        "model": "gpt-4.1-mini",
        "messages": [{"role": "user", "content": "Hello"}],
    },
)
print(response.json())
