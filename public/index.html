Everything in one place

从鉴权到审计，一套网关完成

保留 OpenAI-compatible 调用体验，同时把账户管理能力集中到一个轻量服务中。

K

API Key 管理

为不同用户分配独立凭证，隔离调用权限与账户数据。

B

余额扣费

成功调用自动扣减额度，余额不足时返回明确错误。

L

调用日志

记录路由、模型、状态、扣费和耗时，方便审计排查。

A

管理后台

集中查看用户和日志，创建账户并手动补充余额。

D

用户中心

用户使用自己的 Key 查看余额、凭证和个人调用记录。

O

OpenAI-compatible

兼容常见 Chat Completions 请求格式，降低迁移成本。

Quick start

四步开始调用

不需要复杂集成，从获取凭证到查看日志只需几分钟。

01

创建用户

管理员在后台创建账户并设置初始余额。

02

获取 API Key

保存系统生成的用户凭证，用于后续鉴权。

03

调用接口

通过 Bearer Token 请求 OpenAI-compatible API。

04

查看余额和日志

进入用户中心了解额度与每一次调用状态。
